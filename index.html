Ajuda

Bem-vindo ao CATÁLOGO DE ARTE GNEWS! Siga estas instruções:

  • Clique no botão add para adicionar um novo cartão (Normal ou Post-It).
  • Use a pesquisa ou filtro por tag para encontrar cartões específicos.
  • Clique no nome do cartão ou na imagem para ver detalhes completos (apenas para cartões normais).
  • Assista a tutoriais clicando no ícone play_circle nos detalhes.
  • Acesse pastas clicando no ícone folder nos detalhes.
  • Edite, exclua ou duplique cartões usando os botões nos cartões.
  • Exporte ou importe cartões usando os botões na barra flutuante à direita.
  • Pressione Ctrl + N para adicionar, Ctrl + S ou Enter para salvar e Esc para fechar modais.
